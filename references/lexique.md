## 1. Les 4 types d'unités

Le LÉR manipule quatre types fondamentaux d'unités :

| Type | Notation | Description | Exemples |
|---|---|---|---|
| **Signes** | `sym` | Unité minimale : mot, objet, événement | miroir, seuil, ombre, 1414 |
| **États** | `etat` | Dynamique interne du sujet ou du texte | tension, ouverture, bifurcation |
| **Archétypes** | `arch` | Structures profondes, dynamiques universelles | errant, voyant, tisseur |
| **Gestes** | `geste` | Actions symboliques | inscrire, renverser, ouvrir |

## 2. Table des symboles (sym)

Table des symboles opératoires du LÉR. Chaque symbole est un **point d'interaction**
(pas seulement de représentation) — il agit.

| Symbole | Champ | Sens de base | Usages typiques |
|---|---|---|---|
| `miroir` | sym | reflet, identité, double | travail sur l'image de soi, motif du double |
| `seuil` | sym | passage, transition, ligne rouge | rituels de changement, début de cycle |
| `porte` | sym | accès, choix, invitation | activation de rituel de seuil |
| `ombre` | sym | inconscient, peur, latent | rituels de traversée, travail sur blocages |
| `brume` | sym | confusion, incertitude fertile | repérage de zones de transition |
| `tunnel` | sym | passage difficile, épreuve | navigation en zone de seuil ou d'ombre |
| `lumière` | sym | prise de conscience, révélation | sortie de cycle, clarification |
| `forêt` | sym | complexité, inconnu fertile | navigation symbolique, exploration |
| `route` | sym | trajectoire, destin, direction | analyse de direction de vie |
| `maison` | sym | intériorité, base, identité profonde | archétype du foyer interne |
| `eau` | sym | émotion, inconscient fluide | transduction émotionnelle |
| `feu` | sym | transformation, désir, énergie | rituels de transmutation |
| `cercle` | sym | cycle, répétition, boucle | analyse de boucles, rituel de clôture |

### Polarités dynamiques

Chaque symbole peut fonctionner en polarité positive ou négative selon le contexte :

| Symbole | Polarité + | Polarité − |
|---|---|---|
| `miroir` | Clarté, connaissance de soi | Piège narcissique, boucle |
| `ombre` | Force intérieure cachée | Blocage, peur paralysante |
| `seuil` | Passage intentionnel | Hésitation chronique |
| `feu` | Transformation libératrice | Destruction non maîtrisée |
| `eau` | Fluidité émotionnelle | Submersion, dissolution |

## 3. Les 12 archétypes fondamentaux (arch)

Les archétypes sont des **structures profondes**, des dynamiques universelles qui organisent
l'expérience symbolique. Ils ne sont pas des personnages — ce sont des **forces structurantes**.

| Archétype | Dynamique principale | Cycle associé |
|---|---|---|
| **L'Errant** | Rupture → quête → passage | Fracture → Seuil → Transformation |
| **Le Tisseur** | Connexion, unification, coopération | Stabilisation → Tissage → Expansion |
| **Le Gardien** | Protection, conservation, limite | Fermeture → Scellement → Stabilité |
| **Le Messager** | Transmission, signal, pont | Appel → Passage → Livraison |
| **L'Ombre** | Force latente, refoulé, puissance cachée | Descente → Rencontre → Intégration |
| **L'Éclaireur** | Révélation, exploration, précurseur | Découverte → Cartographie → Retour |
| **L'Enfant-Source** | Origine, potentiel pur, commencement | Naissance → Croissance → Émergence |
| **Le Voyageur** | Mouvement, adaptation, curiosité | Départ → Traversée → Arrivée |
| **Le Témoin** | Observation neutre, mémoire, conscience | Présence → Enregistrement → Transmission |
| **Le Miroir** | Réflexion, dédoublement, résonance | Rencontre → Reflet → Reconnaissance |
| **Le Semeur** | Plantation, intention, geste dans le futur | Semaille → Attente → Récolte |
| **Le Transformeur** | Alchimie, mutation, passage d'état | Dissolution → Transmutation → Cristallisation |

### Utilisation dans le LÉR

Les archétypes sont activés par `RESONE()` et nommés par `SIGNIFIE("nommer [arch]")`.
Chaque archétype porte un **cycle narratif** propre, utilisable pour la prédiction acausale.

## 4. États intérieurs codifiés (etat)

Les états intérieurs décrivent la **dynamique en cours** dans le champ symbolique du sujet.
Ils sont produits par `META()` et peuvent être convertis par `TRANSDUIT()`.

| État | Description | Geste recommandé |
|---|---|---|
| `ouverture` | Passage activé, réceptivité | SIGNIFIE("inscrire seuil") |
| `tension` | Force opposée, résistance active | SIGNIFIE("apaiser tension") |
| `alignement` | Cohérence rétablie, flux harmonieux | SIGNIFIE("sceller cycle") |
| `bifurcation` | Moment de choix, croisement de chemins | SIGNIFIE("orienter passage") |
| `densité` | Accumulation symbolique, surcharge de sens | SIGNIFIE("dissoudre densité") |
| `brume` | Confusion, incertitude, entre-deux | SIGNIFIE("traverser brume") |
| `clarté` | Visibilité, compréhension soudaine | SIGNIFIE("inscrire clarté") |
| `appel` | Signal d'une direction, invitation | SIGNIFIE("nommer appel") |
| `résistance` | Blocage actif, refus symbolique | SIGNIFIE("renverser résistance") |
| `progression` | Mouvement positif, cycle en expansion | SIGNIFIE("stabiliser progression") |
| `instabilité` | Chaos symbolique, perte de cohérence | SIGNIFIE("stabiliser") |
| `émergence` | Naissance d'un nouveau motif | SIGNIFIE("inscrire émergence") |

## 5. Zones symboliques

Les zones organisent l'**espace symbolique** dans lequel se déplace le sujet.

| Zone | Description | Archétypes associés |
|---|---|---|
| **Zone de Seuil** | Passage imminent, frontière critique | Errant, Messager, Voyageur |
| **Zone d'Ombre** | Latent, inconscient, caché | Ombre, Témoin |
| **Zone de Double** | Répétition, miroir, boucle identitaire | Miroir, Errant |
| **Zone de Fracture** | Rupture, bascule, chaos ordonné | Transformeur, Errant |
| **Zone d'Appel** | Signal, vocation, direction émergente | Messager, Éclaireur, Semeur |
| **Zone de Retour** | Cycle accompli, intégration | Voyageur, Témoin, Gardien |

## 6. Cycles archétypaux

La structure cyclique universelle du LÉR comporte 8 phases.
Chaque archétype traverse ce cycle à sa manière.

```
1. Appel       ← le signal se manifeste
2. Ombre       ← rencontre avec le latent
3. Fracture    ← rupture nécessaire
4. Seuil       ← point de passage critique
5. Passage     ← traversée active
6. Transformation ← mutation symbolique
7. Alignement  ← cohérence retrouvée
8. Retour      ← intégration et mémoire
```

### Utilisation en prédiction acausale

La prédiction acausale ne dit pas **ce qui va arriver** — elle révèle **ce qui cherche à arriver**
en identifiant où se situe le sujet dans le cycle archétypal.

```
META(situation) → arch activé + phase du cycle
→ "Le système est en phase Fracture de l'archétype Errant,
   avec un seuil latent dans la Zone d'Ombre."
→ Direction probable : vers le Seuil (phase 4)
```

### Les 4 fondements de la prédiction acausale

| Fondement | Description |
|---|---|
| **Récurrence** | Ce qui revient cherche à se manifester |
| **Résonance** | Ce qui se répond se renforce |
| **Structure archétypale** | Chaque archétype porte un cycle narratif |
| **Tension symbolique** | Les tensions ouvrent les passages |
