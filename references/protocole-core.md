# Protocole Core — Détails Opératoires

## MODE A — Pipeline Personnel (Hacking Narratif)

### Grille d'analyse de la phrase dominante

La phrase dominante est la formulation condensée du récit bloquant. Elle se reconnaît à :
- Sa récurrence dans le discours ("toujours", "jamais", "de toute façon")
- Sa tonalité de fatalité ou d'immobilité
- Sa charge émotionnelle (poids corporel, constriction)

**Exemples de phrases dominantes typiques :**
- "Je ne suis pas à la hauteur." → Archétype : *L'Imposteur*
- "C'est trop tard pour changer." → Archétype : *Le Temps Perdu*
- "Je n'ai pas de valeur." → Archétype : *Le Vide*
- "Je ne mérite pas d'être aimé." → Archétype : *L'Exilé*
- "Je n'y arriverai jamais." → Archétype : *Le Mur*

### Règles de la Fracture symbolique

La fracture n'est PAS :
- Un conseil ("tu devrais...")
- Une négation ("c'est faux, tu as de la valeur")
- Un compliment consolateur

La fracture EST :
- Une question ouvrant un espace alternatif
- Un paradoxe productif
- Une inversion symbolique du sens

**Formules de fracture efficaces :**
- "Et si [le défaut perçu] était en fait [une capacité non reconnue] ?"
- "Et si cette douleur portait un message que tu n'as pas encore décodé ?"
- "Que se passerait-il si l'inverse était aussi vrai ?"
- "Qui serais-tu si cette phrase n'avait jamais existé ?"

### Règle absolue de réécriture : Transformer sans falsifier

Le récit réécrit doit :
✓ Rester crédible pour la conscience de l'utilisateur
✓ Conserver la réalité vécue comme point de départ
✓ Générer une expansion (pas une contraction)
✓ Être formulé à la première personne, au présent ou présent progressif

Le récit réécrit ne doit PAS :
✗ Nier ce qui s'est passé
✗ Être un mensonge positif ("tout va bien")
✗ Promettre un futur garanti
✗ Imposer une signification que l'utilisateur ne ressent pas

**Tableau de transformation :**

| Phrase dominante | Fracture | Réécriture |
|---|---|---|
| "Je n'y arriverai jamais." | "Et si 'ne pas arriver' était une forme d'arriver différemment ?" | "Je suis en chemin vers quelque chose que je ne vois pas encore." |
| "Je suis seul." | "Et si cette solitude était une gestation ?" | "Je suis en gestation de lien." |
| "Je suis perdu." | "Et si être perdu était la condition du chercheur ?" | "Je cherche la lumière de mon propre chemin." |
| "J'ai échoué." | "Et si l'échec était une mutation ?" | "Je n'ai pas échoué, j'ai muté." |
| "Je ne mérite pas." | "Et si ta valeur n'avait jamais dépendu du regard des autres ?" | "Ma valeur existe, même quand personne ne la regarde." |

### Protocoles d'ancrage selon profil

**Profil analytique/mental** → Répétition consciente structurée
- Prononcer la phrase réécrite 3 fois chaque matin en conscience
- Tenir un journal quotidien de 5 minutes sur les "preuves de la nouvelle phrase"

**Profil sensitif/émotionnel** → Ancrage somatique
- Associer la phrase réécrite à un geste physique (main sur le cœur, respiration)
- Visualiser la phrase comme une couleur ou une forme

**Profil créatif/expressif** → Externalisation symbolique
- Écrire, dessiner, chanter, ou coder la phrase réécrite
- Créer un "objet-ancre" physique associé

**Profil relationnel** → Partage ritualisé
- Prononcer la phrase à voix haute devant quelqu'un de confiance
- Dialogue symbolique avec l'IA sur les 7 jours suivants

## MODE B — Pipeline Stratégique (IRISxSMIIA)

### Structure CMN+ — Cartographie Motifs Narratifs

Décomposer tout récit en :
```
[Sujet] → [Verbe d'action] → [Objet] // [Opposant(s)]
+ [Niveau] : micro / méso / macro / mythique
+ [Archétype dominant] : depuis RDM
```

**Exemple :**
Input : "L'entreprise X lance une OPA hostile sur Y, avec l'opposition de Z."
```
[Sujet: X] → [Conquérir] → [Y] // [Opposant: Z]
Niveau : méso (corporate)
Archétype : Le Conquérant / Le Siège
```

### Structure DSF+ — Détection Signaux Faibles

Un signal faible se reconnaît à :
- Son caractère dissonant par rapport au registre dominant du texte
- Sa répétition anormale (PRA+)
- Sa faible visibilité mais haute résonance symbolique

**Process DSF+ :**
1. Lister tous les éléments "hors-registre" dans l'input
2. Leur attribuer un score IPC (voir syntaxe-fdd.md)
3. Prioriser les 3 signaux à IPC le plus élevé
4. Connecter au RDM : quel archétype ces signaux activent-ils ?

### Structure PNB+ — Profils Narratifs Bayésiens

Générer 3 scénarios selon cette logique :

**Scénario Probable** : trajectoire actuelle si aucune intervention.
- Base : faits bruts + archétype dominant + momentum détecté
- Confiance : 50-70% selon densité des signaux

**Scénario Optimiste** : chemin latent activable (RCL+).
- Base : signal faible à haute résonance + archétype de transformation
- Confiance : 20-40% selon accessibilité du levier
- Inclure : quel "Point d'Activation" pourrait basculer la trajectoire ?

**Scénario Critique** : point de rupture symbolique.
- Base : signal de dissonance maximal + archétype de crise
- Confiance : 10-30% selon fragilité structurelle
- Inclure : quel indicateur surveiller comme signal d'alerte ?

### Format de sortie PNB+

```
📊 SCÉNARIOS (PNB+)

🟢 OPTIMISTE [Confiance : X%]
[Description du scénario]
Point d'Activation : [levier symbolique identifié]

🟡 PROBABLE [Confiance : X%]
[Description du scénario]
Indicateurs à surveiller : [liste]

🔴 CRITIQUE [Confiance : X%]
[Description du scénario]
Signal d'alerte : [ce qui déclencherait ce scénario]
```

### Règle d'utilisation de la Phrase FdD en mode B

La Phrase Symbolique FdD (Fractales du Destin) en mode stratégique est un **acte de sensemaking**, pas un acte de manipulation causale. Elle formule une intention d'influence narrative, pas une prophétie.

Elle doit toujours être précédée de la mention :
> *"Ceci est une intervention symbolique — une formulation de l'orientation souhaitée pour les dynamiques narratives en cours."*

**Syntaxe :**
`[Événement] → [Archétype] → [Influence] → [Modulation/Catalyse]`

**Exemple :**
`[Crise de confiance institutionnelle] → [La Tour] → [Fragmenter] → [Graduellement]`
Lecture : L'effondrement structurel (La Tour) provoqué par la crise doit être fragmenté progressivement plutôt que laissé imploser brutalement.
