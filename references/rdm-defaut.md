## OPÉRATEURS NARRATIFS (Fractales du Destin)

Ces opérateurs s'utilisent dans la Phrase FdD comme [Influence] ou [Modulation] :

| Opérateur | Effet narratif | Utilisation |
|---|---|---|
| **Fragmenter** | Diviser pour dissoudre une tension | Sur archétypes de crise |
| **Catalyser** | Accélérer une transformation latente | Sur archétypes de transition |
| **Ancrer** | Stabiliser un récit émergent | Sur archétypes positifs fragiles |
| **Dissoudre** | Effacer un récit toxique progressivement | Sur archétypes négatifs rigides |
| **Relier** | Créer un pont entre deux récits opposés | Sur contextes de polarisation |
| **Amplifier** | Renforcer un signal faible porteur | Sur signaux sous-détectés |
| **Inverser** | Retourner la polarité d'un récit | Sur paradoxes productifs |
| **Suspendre** | Mettre en pause un récit automatique | Sur boucles répétitives |

**Modulations temporelles :**
- *Brusquement* — rupture rapide
- *Graduellement* — transformation lente et profonde
- *Cycliquement* — pattern qui revient
- *En spirale* — transformation non-linéaire, croissante

## NOTES D'ÉVOLUTION DU RDM

Ce fichier est la version par défaut. Pour l'enrichir :

1. Copier ce fichier sous le nom `rdm-custom.md`
2. Ajouter des archétypes selon le format : `[Nom] | [Domaine] | [Polarité] | [Description] | [Modulations]`
3. Les nouveaux motifs détectés lors des sessions apparaissent dans le **bloc MER** sous "Motifs nouveaux à intégrer au RDM"
4. Les intégrer manuellement dans `rdm-custom.md` après validation

Le RDM évolue par accumulation sédimentaire — chaque session dépose un layer.
